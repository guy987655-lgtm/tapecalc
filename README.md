# Tape Calculator

A minimalist dark-mode tape calculator. Pure HTML/CSS/JS in a single `index.html`.

## Features

- Continuous tape history with auto-scroll
- Basic arithmetic (`+`, `-`, `*`, `/`)
- Full keyboard support (digits, operators, Enter, Backspace, Escape)
- Responsive layout: split on desktop, stacked on mobile

## Run locally

Just open `index.html` in a browser — no build step.

## Deploy

Static site. Works on Vercel, Netlify, GitHub Pages, or any static host out of the box.
